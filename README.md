# GUI Calculator (Python Tkinter)

A modern beginner-friendly **GUI Calculator** built using **Python Tkinter**.  
This calculator supports keyboard typing, dark theme UI, and a scrollable calculation history panel.

---

##  Features

 Basic arithmetic operations:  
- Addition (+)  
- Subtraction (-)  
- Multiplication (*)  
- Division (/)  
- Modulus (%)  

 Dark Theme Interface  

 Keyboard Support  
- Type numbers directly  
- Press **Enter** to calculate  
- Press **Backspace** to delete  
- Press **Esc** to clear
